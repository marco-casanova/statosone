# Place your PrusaSlicer .ini profile files here

#

# Examples:

# printer_ender3.ini — exported from PrusaSlicer > Printer Settings > Export

# filament_pla.ini — exported from PrusaSlicer > Filament Settings > Export

#

# To export profiles from PrusaSlicer:

# 1. Open PrusaSlicer

# 2. Go to "Printer Settings" tab → click the save icon → "Export config"

# 3. Repeat for "Filament Settings"

# 4. Place the .ini files in this directory
